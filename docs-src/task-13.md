# Task 13: 한글 자모 연습 모드 개발 (초보자용)

## Description
한글 타이핑 초보자를 위한 자음과 모음 개별 연습 모드를 구현합니다. 이는 '맞춤형 학습 컨텐츠'의 한글 자모 연습 모드 요구사항을 충족합니다.

## Details
한글의 자음(ㅣ, ㅏ, ㅓ...), 모음(ㅏ, ㅑ, ㅓ...) 목록을 정의합니다. `TypingTextDisplay` 컴포넌트를 재활용하거나 자모 연습에 특화된 컴포넌트를 개발하여, 한 번에 하나의 자음 또는 모음을 화면에 표시하고 사용자가 이를 입력하도록 합니다. 입력이 성공하면 다음 자모로 넘어가며, 기본적인 통계(정확성, 소요 시간)를 기록합니다.

## Test Strategy
각 자음과 모음을 정확하게 입력했을 때 다음 자모로 넘어가는지 확인합니다. 오타 발생 시 피드백이 올바르게 나타나고, 이 모드에서 수집된 통계가 다른 연습 모드와 분리되어 기록되는지 테스트합니다.
